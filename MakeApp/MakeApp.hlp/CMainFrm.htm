CMainFrm Module

The CMainFrm Module

The MainFrame object is defined:

class MainFrame : public CMainFrm { ... }

The CMainFrm base module only has the Set App Name and Set Title features. Since the mainframe is an object held in the application data, reaching these functions must be done through theApp object, i.e. CApp.

The application Main Frame module includes several objects which relate to the boarder, menu bar, tool bar and status bar. CMainFrm has functions for setting the application namd and window title but there are other features that are contained in the library. The toolbar is a subclass of CMFCToolBar and provides a toolbar where a wide margin button may be implemented, a drop down menu list and even a combo box. One of the features of managing the toolbar is a message

ON_REGISTERED_MESSAGE(AFX_WM_RESETTOOLBAR, &OnResetToolBar)

The toolbar is actually setup in OnResetToolBar. See the template for details.
